# 🧬 Jeu de la Vie - Python

Simulation graphique du célèbre automate cellulaire de John Horton Conway, implémentée en Python avec Tkinter.

## 📋 Description

Le Jeu de la Vie est un automate cellulaire imaginé par le mathématicien britannique John Horton Conway en 1970. Cette implémentation Python offre une interface graphique intuitive pour observer l'évolution des cellules selon les règles classiques.

## ✨ Fonctionnalités

- **Grille interactive** : Cliquez pour faire vivre ou mourir des cellules
- **Simulation en temps réel** : Observez l'évolution génération après génération
- **Contrôles intuitifs** :
  - ▶️ Démarrer la simulation
  - ⏸️ Mettre en pause
  - 🛑 Quitter l'application
  - 🐌 Régler la vitesse de génération
- **Affichage du numéro de génération**
- **Interface responsive**

## 🛠️ Technologies utilisées

- **Python 3.6+**
- **Tkinter** (inclus dans Python)

## 🎯 Règles du Jeu de la Vie

Chaque cellule évolue selon ces règles simples :
- Une cellule morte avec exactement 3 voisines vivantes devient vivante
- Une cellule vivante avec 2 ou 3 voisines vivantes reste vivante
- Dans tous les autres cas, la cellule meurt ou reste morte
