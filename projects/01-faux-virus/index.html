Simulation de Virus Informatique

Un projet éducatif pour démontrer les risques du téléchargement inconnu

À propos du Projet

Ce projet est une simulation inoffensive d'un virus informatique. Son objectif est d'éduquer les utilisateurs sur les dangers du téléchargement de fichiers provenant de sources non fiables.

Caractéristiques

  • Interface interactive
  • Simulation inoffensive
  • Messages éducatifs

Technologie Utilisée

  • Python
