# Faux Virus - Simulation Éducative

## Description

Ce projet est une simulation inoffensive d'un virus informatique. Son principal objectif est d'éduquer les utilisateurs sur les dangers du téléchargement de fichiers en provenance de sources non fiables.

## Contenu du Projet

- `index.html` - Page principale du projet
- `css/styles.css` - Feuille de style
- `js/script.js` - Code JavaScript
- `assets/` - Ressources (images, etc.)

## Caractéristiques

✓ Interface intuitive et interactive  
✓ Simulation réaliste mais inoffensive  
✓ Messages éducatifs  
✓ Design responsive  
✓ Accessible sur tous les appareils

## Technologie Utilisée

- HTML5
- CSS3  
- JavaScript (Vanilla)

## Comment ouvrir

Double-cliquez simplement sur `index.html` ou ouvrez-le dans votre navigateur préféré.
